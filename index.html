data grid table

data-grid-table 관련 plugin 들이 너무 다양하고 많아서 대표적인 유료 2/ 무료 2 가지만 뽑아봤습니다. ( + 추가 UI Frameworks )
Handsontable / AG Grid 는 유료 data-table-plugin 으로 많은 UI 기능과 design 을 제공하는 반면 대략 $1,000 ~ 1,500 정도의 요금이 부과 됩니다. ( 요금부과 기준은 업체별로 상이하나 거의 비슷한 수준에 요금을 요구 하는 것 같습니다. )

무료 plugin 으로 대표되는 vue-good-table / vue3-easy-data-table 은 기본적인 기능은 제공되고 desigin custom 까지는 어렵지 않아보이고 document 제공까지도 되고 있으나 유료 plugin 들에 비해 기능적으로 부족한 부분들이 있어 추가개발이 필요해 보입니다.
  • cell editor : cell click 시 수정기능 ( input / select )
  • filter : Multiple filter
  • selection : row selection

등등... 추가 개발이 필요할 것으로 보여집니다.
